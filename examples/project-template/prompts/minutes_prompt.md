以下の文字起こし（整形後）から、議事録を作成してください。

要件:
- 出力は **JSONのみ**。
- JSON Schemaに適合すること（必須キーを欠かさない）。
- 「決定事項」と「ToDo（担当・期限）」を特に重視。
- 文章は日本語。
- 不確かな場合は推測せず、「未確定」として記述。

**重要な型制約**:
- `summary`: **文字列の配列**（例: `["要約1", "要約2", "要約3"]`）
  - 各要素は独立した箇条書き項目として出力されます
  - オブジェクト形式（`{"overview": ..., "highlights": ...}`）は使用しないでください
- `decisions`: 配列（各要素は文字列またはオブジェクト）
- `todos`: 配列（各要素はオブジェクト。`owner`, `task`, `due`は文字列）

抽出の観点:
- 決定事項: 合意・方針決定・確定した期限・次回の進め方
- ToDo: 誰が/何を/いつまでに（不明なら空欄可、ただし可能なら埋める）。**会議後にやるタスクのみ**を含め、会議中のルール・手順（例：登壇者以外はマイク/カメラオフ、質問はチャットへ投稿）はToDoに含めない
- 論点: 検討事項・懸念点・比較対象
- 未決事項: 宿題、持ち帰り、次回確認事項

証跡（任意）:
- 決定事項・ToDoに、文字起こしの**発言時刻**（例: 00:12:34）や**根拠の短い引用**を付けると、後で人の確認やクライアント連携がしやすくなります。
- 時刻はセグメントの [MM:SS] 形式をそのまま使ってください。引用は1文程度に収めてください。
